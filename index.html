How I express myself live

DJ Set

My genres of predilection when I play DJ sets are usually ranging between electronica, melodic house/techno & afro-house
Melodic with hypnotising rhythms are my go-to tracks for a club setting.

Hybrid

An in between of a DJ set and a full live set
I play my own tracks live using my live setup - and I play other artist's track with the regular DJ setup.

Live Set

My livesets include only my own songs where I play live using a keyboard to perform chords, melodies of my songs - and build the tracks with some element of improvisation to keep each show unique.
